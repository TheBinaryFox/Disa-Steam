Disa-Steam
==========
A plugin for Disa that adds support for Steam community chat.

Features
--------
**Features supported:**
- Steam chat support.
- SteamGuard/2FA support.

**Features planned:**
- Steam emoticons. *(through :emote: codes)*
- Group chat support.
